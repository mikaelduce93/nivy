## 🔍 MÉTHODOLOGIE DE RECHERCHE APPROFONDIE

### Phase 1: Identification par Zone (Semaine 1-4)

#### Étape 1: Cartographie des Zones

**Pour chaque ville:**
1. Identifier les quartiers/zones fréquentés par les ados
2. Identifier les zones près des écoles
3. Identifier les zones de shopping/loisirs
4. Créer une carte avec toutes les zones

**Outils:**
- Google Maps
- Connaissance locale
- Demandes aux ados/parents

#### Étape 2: Recherche Systématique par Zone

**Méthode: Google Maps par Zone**

Pour chaque zone identifiée:

1. **Ouvrir Google Maps**
2. **Rechercher par zone + type:**
   - "Restaurants Maarif Casablanca"
   - "Magasins vêtements Gauthier"
   - "Clubs sportifs Hay Riad Rabat"
   - "Cafés Centre-ville Marrakech"

3. **Parcourir TOUS les résultats:**
   - Ne pas se limiter aux premiers résultats
   - Scroller jusqu'au bout
   - Cliquer sur "Afficher plus de résultats"
   - Utiliser les filtres (note, horaires, etc.)

4. **Pour chaque établissement:**
   - Vérifier si c'est local/indépendant (pas une chaîne internationale)
   - Vérifier note (> 3.5/5 minimum)
   - Vérifier présence Instagram/Facebook
   - Noter toutes les informations

**Objectif:** 20-30 établissements par zone × 10-15 zones = 200-450 établissements par ville

#### Étape 3: Recherche Instagram par Zone

**Méthode: Hashtags Locaux**

Pour chaque zone:

1. **Rechercher hashtags locaux:**
   - #maarifcasablanca
   - #gauthiercasablanca
   - #agdalrabat
   - #guelizmarrakech
   - etc.

2. **Explorer les posts:**
   - Identifier les établissements tagués
   - Vérifier si c'est un business local
   - Noter le nom et le type

3. **Suivre les comptes:**
   - Suivre les comptes d'établissements locaux
   - Explorer leurs followers (autres établissements)
   - Explorer les comptes qu'ils suivent

**Objectif:** 50-100 établissements supplémentaires par ville

#### Étape 4: Recherche Facebook par Zone

**Méthode: Pages Business Locales**

1. **Rechercher "Pages" sur Facebook:**
   - Filtrer par ville et type
   - Explorer les pages business locales

2. **Explorer les événements:**
   - Voir quels établissements organisent des événements
   - Identifier les partenaires d'événements

**Objectif:** 30-50 établissements supplémentaires par ville

## 🛠️ OUTILS & TECHNIQUES AVANCÉES

### Technique 1: Google Maps - Recherche par Rue

**Méthode:**
1. Ouvrir Google Maps
2. Zoomer sur une rue spécifique
3. Cliquer sur chaque établissement
4. Vérifier si local/indépendant
5. Noter les informations

**Avantage:** Découvre des établissements non listés dans les recherches

### Technique 2: Instagram - Exploration en Profondeur

**Méthode:**
1. Trouver un compte d'établissement local
2. Explorer ses followers (autres établissements)
3. Explorer les comptes qu'il suit
4. Explorer les hashtags qu'il utilise
5. Répéter pour chaque nouvel établissement trouvé

**Avantage:** Découvre le réseau d'établissements locaux

### Technique 3: Facebook - Groupes Locaux

**Méthode:**
1. Rejoindre des groupes locaux (ex: "Casablanca Restaurants", "Rabat Shopping")
2. Explorer les posts et commentaires
3. Identifier les établissements mentionnés
4. Contacter directement les propriétaires

**Avantage:** Accès direct aux propriétaires

### Technique 4: Visites Terrain (Optionnel mais Efficace)

**Méthode:**
1. Se rendre dans une zone
2. Marcher dans les rues principales
3. Noter tous les établissements vus
4. Prendre des photos des vitrines
5. Noter les coordonnées affichées

**Avantage:** Découvre des établissements non digitaux

### Technique 5: Références Locales

**Méthode:**
1. Demander aux ados/parents de l'app
2. Créer un formulaire de suggestions
3. Récompenser les suggestions (XP bonus)
4. Vérifier et ajouter les suggestions

**Avantage:** Découvre les établissements vraiment fréquentés

## 🎯 CRITÈRES DE SÉLECTION - TOUS TYPES

### Critères Minimum

1. **Type d'Établissement** ☐
   - Local/Indépendant
   - Chaîne internationale
   - Grande chaîne marocaine
   - Franchise
   - (Tous types acceptés)

2. **Pertinence Ados** ☐
   - Public cible inclut 13-18 ans
   - Produits/services adaptés
   - Prix accessibles (ou prix moyens avec discount possible)

3. **Localisation** ☐
   - Zone fréquentée par ados
   - Accessible transport public
   - Près écoles (optionnel mais bonus)

4. **Qualité Minimum** ☐
   - Note > 3.5/5 (si disponible)
   - Pas de réputation très négative
   - Présence digitale (Instagram/Facebook minimum, ou site web)

5. **Potentiel Partenariat** ☐
   - Ouverture au digital (même basique)
   - Intérêt pour développer clientèle
   - Possibilité de négociation (varie selon type)

### Critères Bonus

- Note > 4.0/5
- Reviews > 20
- Présence active réseaux sociaux
- Site web professionnel
- Déjà des offres/promotions
