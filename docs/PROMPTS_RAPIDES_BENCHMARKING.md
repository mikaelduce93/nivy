## 🛍️ RETAIL (MAGASINS)

### Prompt Court

```
Liste 30 magasins au Maroc (Casablanca, Rabat, Marrakech) populaires chez 
les adolescents 13-18 ans: vêtements (Zara, H&M), électronique (Fnac), 
cosmétiques (Sephora), sport (Decathlon). Pour chaque: nom, ville, adresse, 
site web, note Google, prix, pertinence ados.
```

### Prompt par Catégorie

**Vêtements:**
```
Top 20 magasins de vêtements adolescents à Casablanca: nom, adresse, note, prix, 
pourquoi les ados y vont.
```

**Électronique:**
```
Magasins électronique/tech accessibles ados au Maroc: smartphones, gaming, 
casques. Nom, ville, prix, note.
```

**Cosmétiques:**
```
Parfumeries et magasins beauté pour adolescentes au Maroc: nom, ville, prix, 
note, pertinence.
```

## 📊 ÉVALUATION RAPIDE

### Prompt d'Évaluation

```
Évalue ce partenaire sur 100 points:
- Nom: [NOM]
- Type: [TYPE]
- Note Google: [NOTE]
- Reviews: [NOMBRE]

Critères:
1. Pertinence ados (25 pts)
2. Qualité/réputation (20 pts)
3. Valeur commerciale (20 pts)
4. Accessibilité (15 pts)
5. Alignement brand (10 pts)
6. Engagement (10 pts)

Score total et recommandation.
```

## 💬 PROMPTS CONVERSATIONNELS

### Pour ChatGPT/Claude

**Début de conversation:**
```
Je cherche des partenaires pour une app adolescents au Maroc. Peux-tu m'aider 
à identifier des magasins, restaurants, clubs et centres de formation 
pertinents ?
```

**Suite:**
```
Parfait, maintenant évalue chacun selon ces critères: pertinence ados, qualité, 
prix, localisation. Donne un score sur 100.
```

**Finalisation:**
```
Classe les top 20 par priorité et explique pourquoi.
```
