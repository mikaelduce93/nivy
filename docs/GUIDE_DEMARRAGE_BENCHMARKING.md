# 🚀 GUIDE DE DÉMARRAGE RAPIDE - BENCHMARKING PARTENAIRES

## 📋 CHECKLIST DE DÉMARRAGE

### Étape 1: Setup Initial (30 minutes)

- [ ] **Créer la table dans la base de données**
  ```bash
  # Exécuter le script SQL
  psql -d votre_db -f scripts/115_create_partner_prospects_table.sql
  ```

- [ ] **Créer un Google Sheet pour le suivi**
  - Copier le template depuis `docs/TEMPLATE_FICHE_EVALUATION_PARTENAIRE.md`
  - Créer une feuille Google Sheets avec les colonnes nécessaires
  - Partager avec l'équipe

- [ ] **Lire la stratégie complète**
  - Lire `docs/STRATEGIE_BENCHMARKING_PARTENAIRES.md`
  - Comprendre les critères de scoring
  - Identifier les villes prioritaires

### Étape 2: Première Recherche (2-3 heures)

**Objectif:** Identifier 20-30 prospects pour tester le processus

#### Recherche Google Maps

1. **Ouvrir Google Maps**
2. **Rechercher par ville + type:**
   - "Casablanca restaurants adolescents"
   - "Rabat clubs sportifs"
   - "Marrakech cinémas"
   - etc.

3. **Pour chaque résultat intéressant:**
   - Ouvrir la fiche
   - Noter: nom, adresse, téléphone, site web
   - Vérifier note et nombre de reviews
   - Ajouter à la liste

#### Recherche Instagram

1. **Rechercher hashtags:**
   - #casablancateen
   - #marrakechadolescent
   - #rabatjeunes
   - etc.

2. **Explorer les comptes:**
   - Vérifier si c'est un business local
   - Noter le nom et le type
   - Ajouter à la liste

### Étape 3: Première Évaluation (2-3 heures)

**Objectif:** Évaluer les 20-30 prospects identifiés

1. **Pour chaque prospect:**
   - Ouvrir le template de fiche d'évaluation
   - Remplir les informations générales
   - Évaluer chaque critère
   - Calculer le score total

2. **Classer par score:**
   - Identifier les top 10
   - Prioriser les contacts

### Étape 4: Premier Contact (1-2 heures)

**Objectif:** Contacter les top 10 pour tester le processus

1. **Préparer l'email:**
   - Utiliser le template dans la stratégie
   - Personnaliser pour chaque partenaire
   - Présenter clairement Teens Party Morocco

2. **Envoyer les emails:**
   - Suivre les envois
   - Noter les réponses

3. **Analyser les résultats:**
   - Taux de réponse
   - Intérêt exprimé
   - Ajuster le processus si nécessaire
