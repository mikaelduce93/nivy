# Sound library

The 26 sound effects under `public/sounds/*.wav` are **placeholder synth
sounds** generated by `scripts/audio/generate-sounds.mjs`. They exist so the
juice system in `lib/sounds/sound-manager.ts` and `lib/hooks/use-juice.ts`
has audible feedback today; a future authored sound design pass can replace
them file by file without touching any code path.

## Replacing with authored sounds

The file extensions in `lib/sounds/sound-manager.ts > SOUND_CONFIGS` and in
`components/gamification/celebration-overlay.tsx` are `.wav`. To swap in
authored MP3 / OGG assets:

1. Drop the new file at `public/sounds/<name>.<ext>`.
2. Update the corresponding entry in `SOUND_CONFIGS` (just the path).
3. (Optional) Remove the old `.wav` from `public/sounds/`.

The juice system gates everything through three switches (reduced motion,
the `nivy.audio` localStorage mute, per-call `silent`). Asset replacement
does not require touching any consumer.

## Regenerating placeholders

```bash
node scripts/audio/generate-sounds.mjs
```

Edit the `sounds` table at the bottom of the script to tweak frequencies,
shapes, durations or chord stacks. The helpers (`tone`, `noise`, `mix`,
`concat`, `envelope`) cover almost every UI sound shape and the script is
under 200 lines.
