# Favicons

Source of truth for the NIVY favicon mark is the **panda head** as drawn in
[`components/brand/panda-logo.tsx`](../../components/brand/panda-logo.tsx)
(`<PandaIcon expression="happy" />`). It is intentionally geometric — circles
and arcs only — so it survives down to 16x16 without anti-aliasing mush.

## What ships today

| Asset | Path | Notes |
| --- | --- | --- |
| SVG favicon (modern browsers) | `public/icons/panda-favicon.svg` | Vector, no rasterization, used first by `<link rel="icon">`. |
| Safari pinned-tab mask | `public/icons/safari-pinned-tab.svg` | Mono-colour silhouette, `mask-icon color` set to brand purple `#a855f7`. |
| `apple-touch-icon` 180x180 | `public/icons/apple-touch-icon.png` | Generated by `scripts/icons/generate-favicons.mjs`. |
| Browser 16x16 | `public/icons/icon-16x16.png` | Same script. |
| Browser 32x32 | `public/icons/icon-32x32.png` | Same script. |
| iPad 152x152 | `public/icons/icon-152x152.png` | Same script. |
| Android 192x192 | `public/icons/icon-192x192.png` | Same script. |
| Android 512x512 | `public/icons/icon-512x512.png` | Same script. |
| Open Graph 1200x630 | `public/og-image.jpg` | Same script (panda + wordmark on dark gradient). |

`app/layout.tsx > metadata.icons` already references all sizes; the PNGs are
picked up automatically.

## Regenerating

The script depends on `sharp` (already a transitive dependency of Next.js)
and reads the canonical SVG from `public/icons/panda-favicon.svg`.

```bash
node scripts/icons/generate-favicons.mjs
```

It rasterizes the SVG at 4× density per target size, then resizes down with
sharp's high-quality kernel. The OG image is composed inline in the script
(panda + wordmark on a dark vertical gradient) and rendered to JPEG @ 88.

## Visual specs

- Background: rounded square `#0f0f1a` (matches `panda-favicon.svg` `<rect>`).
  Keeps the icon legible on both white (Chrome) and dark (browser PWA
  shortcut) surfaces.
- Brand purple accent: `#a855f7` (`oklch(0.65 0.20 290)` family) — used for
  the Safari pinned-tab `mask-icon color` and any future macOS dock badge.
- The SVG is **theme-aware via `currentColor`** when used inline in React
  (`<PandaIcon />`). For the favicon the colors are baked-in (CSS context is
  not available inside `<link rel="icon">`).

## Manifest
