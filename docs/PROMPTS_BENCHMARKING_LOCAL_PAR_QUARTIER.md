# 🗺️ PROMPTS BENCHMARKING PAR QUARTIER
## Prompts spécifiques pour identifier les établissements locaux quartier par quartier

---

## 🎯 STRATÉGIE: QUARTIER PAR QUARTIER

Au lieu de chercher par type d'établissement, chercher **par quartier/zone** pour découvrir TOUS les établissements locaux d'un coup.

**Avantage:** Découvre des établissements qu'on aurait manqués en cherchant par type.

---

## 🏙️ CASABLANCA

### Quartier Maarif

```
Identifie TOUS les établissements (LOCAUX, CHAÎNES, FRANCHISES) dans le 
quartier Maarif, Casablanca, qui seraient pertinents pour des adolescents 
13-18 ans.

QUARTIER: Maarif, Casablanca
ZONE: Quartier commercial et résidentiel, fréquenté par jeunes et familles

TYPES À IDENTIFIER:
1. RETAIL: Boutiques vêtements (Zara, H&M, locales), accessoires, bijoux, 
   électronique (Fnac, locales), cosmétiques (Sephora, locales), livres, sport 
   (Decathlon, Nike, locales)
2. VENUE: Restaurants (McDonald's, Burger King, locaux), cafés (Starbucks, 
   locaux), boulangeries, glaciers, food trucks, cinémas (Megarama, etc.)
3. CLUB: Clubs sportifs, danse, musique, art, coding (tous types)
4. EDUCATION: Soutien scolaire, langues, coding (tous types)

INCLURE TOUS LES TYPES:
- Chaînes internationales (Zara, H&M, McDonald's, Starbucks, Fnac, Sephora, 
  Decathlon, Megarama, etc.)
- Grandes chaînes marocaines
- Franchises
- Établissements locaux et indépendants
- Tous les établissements pertinents

POUR CHAQUE ÉTABLISSEMENT:
- Nom exact
- Type (retail/venue/club/education)
- Sous-type (restaurant, boutique vêtements, etc.)
- Type d'établissement: Local/Indépendant / Chaîne internationale / Grande chaîne marocaine / Franchise
- Adresse complète (rue précise)
- Téléphone
- Instagram
- Facebook
- Site web (si disponible)
- Note Google (si disponible)
- Reviews (si disponible)
- Prix moyens
- Pourquoi pertinent pour ados

GÉNÈRE UNE LISTE EXHAUSTIVE de TOUS les établissements locaux pertinents dans 
Maarif. Ne manque aucun établissement.
```

## 🔍 PROMPT GÉNÉRIQUE PAR QUARTIER

### Template Réutilisable

```
Identifie TOUS les établissements (LOCAUX, CHAÎNES, FRANCHISES) dans 
[QUARTIER], [VILLE], Maroc, qui seraient pertinents pour des adolescents 
13-18 ans.

QUARTIER: [QUARTIER]
VILLE: [VILLE]
CONTEXTE: [Description du quartier - zones fréquentées par jeunes, près écoles, etc.]

TYPES À IDENTIFIER (TOUS TYPES):
1. RETAIL: 
   - Chaînes: Zara, H&M, Pull & Bear, Fnac, Sephora, Decathlon, Nike, Adidas, etc.
   - Grandes chaînes marocaines
   - Boutiques locales: vêtements, accessoires, bijoux, électronique, cosmétiques, 
     livres, sport, chaussures, gaming
2. VENUE: 
   - Chaînes: McDonald's, Burger King, Starbucks, Megarama, etc.
   - Grandes chaînes marocaines
   - Établissements locaux: restaurants, cafés, boulangeries, glaciers, food trucks, 
     cinémas, bowling, escape rooms, laser tag, salles de jeux
3. CLUB: 
   - Grandes chaînes (si présentes)
   - Clubs locaux: sportifs, danse, musique, art, coding, fitness, arts martiaux
4. EDUCATION: 
   - Grandes chaînes (si présentes)
   - Centres locaux: soutien scolaire, langues, coding, prépa examens

INCLURE TOUS LES TYPES:
- Chaînes internationales
- Grandes chaînes marocaines
- Franchises
- Établissements locaux et indépendants
- Tous les établissements pertinents pour ados

POUR CHAQUE ÉTABLISSEMENT, FOURNIR:
1. Nom exact de l'établissement
2. Type: retail / venue / club / education
3. Sous-type (ex: restaurant, boutique vêtements, club football, etc.)
4. Type d'établissement: Local/Indépendant / Chaîne internationale / Grande chaîne marocaine / Franchise
5. Adresse complète (rue précise, numéro si disponible)
6. Téléphone
7. Instagram (si disponible)
8. Facebook (si disponible)
9. Site web (si disponible)
10. Note Google (si disponible)
11. Nombre de reviews (si disponible)
12. Prix moyens: budget / moyen / haut de gamme
13. Pourquoi pertinent pour ados (1-2 phrases)
14. Potentiel de partenariat: 1-5 (5 = excellent) - Note: peut varier selon type

MÉTHODE:
- Utilise Google Maps pour cette zone spécifique
- Explore Instagram avec hashtags locaux (#[quartier][ville])
- Explore Facebook pages business de cette zone
- Sois exhaustif - ne manque aucun établissement local

GÉNÈRE UNE LISTE EXHAUSTIVE de TOUS les établissements locaux pertinents 
dans [QUARTIER], [VILLE]. Objectif: 20-50 établissements par quartier selon 
la taille.
```

## 📊 UTILISATION

### Pour chaque ville:

1. **Identifier les quartiers prioritaires** (10-15 quartiers par grande ville)
2. **Utiliser le prompt générique** pour chaque quartier
3. **Compiler les résultats** dans une base de données
4. **Vérifier les doublons** et nettoyer
5. **Évaluer chaque établissement** avec la fiche d'évaluation

### Objectif par quartier:

- **Petit quartier:** 10-20 établissements
- **Quartier moyen:** 20-30 établissements
- **Grand quartier:** 30-50 établissements
