# 🤖 PROMPTS DE BENCHMARKING PAR TYPE DE PARTENAIRE
## Prompts prêts à l'emploi pour identifier et évaluer les partenaires

---

## 📋 UTILISATION

Ces prompts peuvent être utilisés avec:
- **ChatGPT / Claude / autres IA** - Pour générer des listes de prospects
- **Recherche web** - Pour guider vos recherches Google
- **Équipe** - Pour standardiser le processus de recherche

**Format:** Copiez-collez le prompt et adaptez les variables [VILLE], [TYPE], etc.

## 🛍️ PROMPT 1: RETAIL (MAGASINS - TOUS TYPES)

### ✅ STRATÉGIE: INCLURE TOUS LES TYPES

**INCLURE:** 
- Chaînes internationales (Zara, H&M, Pull & Bear, Fnac, Sephora, Decathlon, etc.)
- Grandes chaînes marocaines
- Boutiques locales et indépendantes
- Franchises
- Tous les magasins pertinents pour ados

### Prompt Principal - Tous Types

```
Je cherche à identifier TOUS les magasins et boutiques au Maroc (LOCAUX, 
CHAÎNES, FRANCHISES) qui seraient d'excellents partenaires pour une plateforme 
destinée aux adolescents (13-18 ans).

CONTEXTE:
- Plateforme: Teens Party Morocco - application pour adolescents marocains
- Objectif: Offrir des réductions exclusives aux membres VIP
- Public cible: Adolescents 13-18 ans
- Stratégie: Identifier TOUS les types d'établissements pertinents (locaux + chaînes)

CRITÈRES DE RECHERCHE:
1. Magasins populaires chez les adolescents marocains
2. Prix accessibles pour un budget adolescent (ou prix moyens avec discount possible)
3. Localisation dans les zones fréquentées par les jeunes (quartiers, près écoles, centres commerciaux)
4. Bonne réputation (note > 3.5/5 minimum)
5. Présence sur Instagram/Facebook ou site web
6. Ouverture au digital et aux partenariats (varie selon type)

TYPES DE MAGASINS À IDENTIFIER (TOUS TYPES):
- Chaînes internationales: Zara, H&M, Pull & Bear, Bershka, Stradivarius, Mango, 
  Massimo Dutti, Fnac, Media Markt, Sephora, MAC, Decathlon, Nike, Adidas, etc.
- Grandes chaînes marocaines: Toutes les chaînes marocaines pertinentes
- Boutiques locales et indépendantes: streetwear, vintage, créateurs locaux
- Boutiques d'accessoires et bijoux (chaînes et locales)
- Boutiques d'électronique (chaînes et indépendantes)
- Parfumeries et boutiques de cosmétiques (chaînes et locales)
- Librairies (chaînes et indépendantes)
- Boutiques de sport (chaînes et locales)
- Boutiques de chaussures (chaînes et indépendantes)
- Boutiques gaming (chaînes et locales)
- Tous les commerces pertinents pour ados

INCLURE TOUS LES TYPES:
- Chaînes internationales
- Grandes chaînes marocaines
- Franchises
- Boutiques locales et indépendantes
- Tous les établissements pertinents

POUR CHAQUE MAGASIN IDENTIFIÉ, FOURNIR:
1. Nom exact de l'entreprise
2. Type de magasin (vêtements, électronique, etc.)
3. Type d'établissement: Local/Indépendant / Chaîne internationale / Grande chaîne marocaine / Franchise
4. Ville et quartier précis
5. Adresse complète
6. Téléphone
7. Site web (si disponible)
8. Instagram / Facebook
9. Note moyenne Google (si disponible)
10. Nombre de reviews (si disponible)
11. Gamme de prix (budget, moyen, haut de gamme)
12. Pourquoi c'est pertinent pour les ados
13. Potentiel de partenariat (1-5) - Note: peut varier selon type

ZONES PRIORITAIRES PAR VILLE:
- Casablanca: Maarif, Gauthier, Anfa, Bourgogne, Centre-ville, Hay Riad, Morocco Mall, etc.
- Rabat: Agdal, Hay Riad, Centre-ville, Akkari
- Marrakech: Guéliz, Hivernage, Centre-ville
- Fès: Ville Nouvelle, Centre-ville
- Tanger: Centre-ville, Malabata
- Agadir: Centre-ville, Hay Mohammadi

GÉNÈRE UNE LISTE EXHAUSTIVE de 100-150 MAGASINS (tous types) avec toutes ces 
informations, incluant chaînes internationales, chaînes marocaines, et établissements locaux.
```

### Prompt de Recherche Spécifique par Catégorie

## 📊 PROMPT 6: ÉVALUATION & SCORING

### Prompt d'Évaluation Complète

```
Évalue ce partenaire potentiel selon les critères suivants et calcule 
un score sur 100 points:

PARTENAIRE:
- Nom: [NOM]
- Type: [TYPE]
- Ville: [VILLE]
- Site web: [SITE]
- Note Google: [NOTE]
- Reviews: [NOMBRE]

CRITÈRES D'ÉVALUATION:

1. PERTINENCE POUR L'AUDIENCE (25 pts)
   - Public cible aligné 13-18 ans: ___/10
   - Produits/services adaptés: ___/8
   - Localisation accessible: ___/7

2. QUALITÉ & RÉPUTATION (20 pts)
   - Note moyenne: ___/8
   - Nombre de reviews: ___/5
   - Réputation en ligne: ___/5
   - Certifications: ___/2

3. VALEUR COMMERCIALE (20 pts)
   - Discount proposé: ___% → ___/8
   - Commission acceptée: ___% → ___/6
   - Volume clients mensuel: ___ → ___/6

4. ACCESSIBILITÉ (15 pts)
   - Proximité zones jeunes: ___/8
   - Transport public: ___/4
   - Parking: ___/3

5. ALIGNEMENT BRAND (10 pts)
   - Image de marque: ___/5
   - Valeurs alignées: ___/5

6. POTENTIEL ENGAGEMENT (10 pts)
   - Activité réseaux sociaux: ___/5
   - Ouverture digital: ___/5

CALCULE:
- Score total: ___/100
- Catégorie: Excellent (85-100) / Très Bon (70-84) / Bon (55-69) / 
  Moyen (40-54) / Faible (<40)
- Recommandation: Priorité MAXIMALE / HAUTE / MOYENNE / BASSE / Éliminer

PROVIDE une évaluation détaillée avec justifications pour chaque score.
```

## 🎯 PROMPT 7: COMPARAISON & SÉLECTION

### Prompt de Comparaison

```
Compare ces [N] partenaires du même type ([TYPE]) et classe-les par 
ordre de priorité pour un partenariat:

PARTENAIRES:
[Liste des partenaires avec leurs informations]

CRITÈRES DE COMPARAISON:
1. Score global (pertinence + qualité + valeur)
2. Conditions commerciales (discount, commission)
3. Potentiel volume
4. Alignement stratégique
5. Facilité d'intégration

POUR CHAQUE PARTENAIRE:
- Score global
- Points forts
- Points faibles
- Recommandation

CLASSE les partenaires par ordre de priorité (1 = meilleur) et explique 
pourquoi chaque partenaire est classé à cette position.
```
